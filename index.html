WikiMe

WikiMe is a project that allows it's users to generate a random multiple choice question based off of a topic that they select. It does this by scrapping a wikipedia page of that topic (hence the name WikiMe) and creating questions based off of it's content.
I worked on the front end of this project using technologies such as Javascript, React, and Chakra UI. This project was developed for HackTheNorth, and in a group of three, where we were able to collaboratively finish this project in under 36 hours.

PCB Design and Engineering

Torchic is a temperature reading embedded circuit that is detrimental to the performance of Waterloop's (the University of Waterloo's Hyperloop Team) hyperloop pod. It ensures that the temperature of key components can be measured in order to be regulated.
I designed and created this PCB using a tool called Altium Designer. By taking on this project, I gained many valuable skills, such as soldering, testing, and designing PCBs.
